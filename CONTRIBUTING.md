# Contributing to promptlint-cli

Thank you for your interest in improving `promptlint-cli`! We welcome community contributions, especially new linting rules.

## How to Add a New Rule

1. Fork the repository and create your feature branch (`git checkout -b feature/my-new-rule`).
2. Add your regex pattern and rule logic to `promptlint/rules.py`.
3. Add corresponding unit tests to `tests/test_rules.py`.
4. Ensure all tests pass by running `pytest`.
5. Open a Pull Request outlining the utility of the rule.
